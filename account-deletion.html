타닥타닥 계정 삭제 및 데이터 삭제 안내

Published: 2026-03-06 Independent URL

Quick Summary

  • 앱 내 `프로필 > 계정 관리 > 삭제하기` 경로에서 계정 삭제를 직접 진행할 수 있습니다.
  • 삭제가 확정되면 현재 서비스 기준으로 프로필, 기록, 작성 글, 좋아요/가져오기/신고 내역이 삭제됩니다.
  • 앱 접근이 어려우면 omilsoft@gmail.com 으로 문의할 수 있습니다.

1. 삭제 요청 방법

  1. 타닥타닥 앱 실행
  2. `프로필 > 계정 관리`로 이동
  3. `삭제하기` 선택
  4. `삭제 안내 확인` 후 최종 `계정 삭제` 확인

앱 접근이 어려운 경우에는 지원 이메일로 계정 삭제 관련 도움을 요청할 수 있습니다.

2. 삭제 및 익명화 범위

  • 삭제 대상: 프로필, 연습 기록, 사용자 작성 글, 커뮤니티 공개 글, 좋아요, 가져오기, 신고 내역, 관련 감사 기록
  • 게스트 계정의 경우: 게스트 계정과 로컬 기록, 작성 글이 함께 제거됩니다.
  • 현재 서비스 기준에서는 위 항목을 별도 익명화 보존하지 않고 삭제합니다.

3. 처리 방식과 SLA

  • 앱 내에서 최종 확인된 삭제는 즉시 수행되며, 삭제 후에는 복구할 수 없습니다.
  • 이메일 문의는 영업일 기준 순차적으로 확인하고 회신합니다.
  • 본인 확인이 필요한 경우 추가 확인 요청을 드릴 수 있습니다.

4. 참고 사항

  • 타닥타닥 계정 삭제는 앱 내 데이터와 서비스 계정을 대상으로 합니다.
  • 외부 로그인 사업자 계정 자체의 해지나 삭제는 해당 사업자 정책을 따릅니다.
